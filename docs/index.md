
# Download the dataset
Click [this link](https://github.com/Goli-SF/uzh_demo/blob/main/data/moma_artworks.zip) to download the dataset. Unzip the downloaded file on your computer.

# Download OpenRefine
Click [this link](https://openrefine.org/download) to download OpenRefine. Unzip the downloaded file on your computer and then run `openrefine.exe`.
