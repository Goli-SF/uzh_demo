
# Download the dataset
Click [this link](https://raw.githubusercontent.com/Goli-SF/uzh_demo/main/data/moma_artworks.zip) to download the dataset. Unzip the downloaded file on your computer.

# Download OpenRefine
Click [this link](https://openrefine.org/download) to download OpenRefine. Unzip the downloaded file on your computer and then run `openrefine.exe`.


Geospatial analysis with [Kepler.gl](https://kepler.gl/)

Image annotation with [Make Sense](https://www.makesense.ai/?utm_source=chatgpt.com)
or (VGG Image Annotator)[https://gitlab.com/vgg/via/-/blob/master/README.md?ref_type=heads&utm_source=chatgpt.com]
